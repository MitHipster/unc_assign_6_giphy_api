# Movie GIFs
UNC Boot Camp Assignment 6 Giphy API

[GitHub Page](https://mithipster.github.io/unc_assign_6_giphy_api/)

Movie GIFs site that uses the GIPHY API to make a dynamic web page that populates with gifs by clicking on movie title buttons. Users can also add their own favorite movie titles using the provided input field. Gifs are first loaded as still images, but become animated when the image is clicked; subsequent clicks toggle the animation off and on.

## Technology
Movie GIFs was created with the following tools:
  * jQuery
  * AJAX
  * Giphy API
